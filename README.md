# Concentration-game-2025

4. **Concentration (Memory Game)**
    - Use cards
    - Will need to display "wrong" guess until a timer expires.
    - Must lose after after X number of bad guesses.

Cards appear on screen, face down, with pairs. 
When you select a card, it flips face up. 
Select a second card.
If matches, then they stay face up and you continue.
If not, then return face down. try again. 
Game won when all cards face up
Game over when lives expire.

Ice box
- Choose varying numbers of cards to increase difficulty
- themed cards/background 
- random bonus for completing a specific task 
    - task: correct on first try, complete specific set next
    - reward: pause timer, reset timer, flash all cards, one card permanently up
- continuous mode, where you have a set amount of lives and the number of cards keeps increasing
